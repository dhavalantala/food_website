# Responsive Food Delivery Website

A responsive food delivery website called **Foodie** — built with pure HTML, CSS, and JavaScript. It allows users to browse a food menu, add items to a cart, and experience a smooth, modern UI across all devices.

## 🌐 Live Demo

[https://dhavalantala.github.io/food_website/](https://dhavalantala.github.io/food_website/)

## 🛠️ Tech Stack

- **HTML5** — Semantic page structure and layout
- **CSS3** — Styling, Flexbox-based responsive design, animations and hover effects
- **JavaScript** — Add to cart functionality, dynamic cart total, and UI interactions

## 📄 Pages & Sections

- **Home** — Hero section with a call-to-action "Order Now" button
- **How It Works** — Three-step service overview (Easy to Order, Best Quality, Fast Delivery)
- **Menu** — Most popular food items with Add to Cart buttons
- **Cart** — Slide-in cart panel showing selected items and total price
- **Reviews** — Customer testimonials with profile photos
- **Mobile App** — Promotional section for app download
- **Newsletter** — Email subscription section
- **Footer** — Navigation links (Menu, Company, Support)

## ✨ Features

- Fully responsive across mobile, tablet, and desktop
- Interactive shopping cart with real-time total calculation
- Smooth UI transitions and hover effects
- Mobile-friendly navigation with hamburger menu
